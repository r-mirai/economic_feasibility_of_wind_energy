\section{ДАННЫЕ ЭНЕРГОПОТРЕБЛЕНИЯ}
\subsection{Получение данных энергопотребления}

Данные энергопотребления в данной работе формируются эмпирическим способом.
Для этого сутки были разделены на четыре временных периода: утро, день, вечер и ночь.
Для каждого периода была выполнена оценка используемых электроприборов, их мощности и суммарной продолжительности работы.

На основе полученных данных был разработан алгоритм формирования суточных профилей энергопотребления.
Для каждого электроприбора суммарное время его работы в пределах соответствующего периода сохраняется, однако конкретные часы включения определяются случайным образом.

При этом распределение работы электроприборов по часам внутри каждого периода является неравномерным.
Для формирования более реалистичных профилей энергопотребления используется вероятностное распределение, которое позволяет сгруппировать нагрузку во времени.
Такой подход отражает характерное поведение потребителей, при котором использование электроприборов, как правило, сосредоточено в определённые часы суток.

Использование вероятностного распределения позволяет сохранить средние значения энергопотребления и одновременно учесть кратковременные пики нагрузки, возникающие при одновременной работе нескольких электроприборов.
При формировании данных также учитывается циклический характер времени суток, что особенно важно для ночного периода.
